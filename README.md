# Solas-Sense-Shamrock
![jasonsmacropad](https://github.com/user-attachments/assets/77b8809e-896d-4c6f-8231-1e01283b7130)


A LED Lamp shaped like a shamrock that changes led color based off the room brightness.
I Named it Solas Sense so I can make different designs and Keep it simply orginized.

### My Experience
This was an easier project than most but its still fun and unquie. 
I struggled on the Designing as its an odd shape compared to just squares and a circle or two.

### Parts
- 1x Esp32-C3 SuperMini
- 3x RGB Leds Anode
- 1x Light Dependent Resistors
- 9x 220Omh Resistors
- 1x 10kOmh Resistor

- Top-Case.stl
- Bottom-Case.stl
- SolasSense-Code.ino
